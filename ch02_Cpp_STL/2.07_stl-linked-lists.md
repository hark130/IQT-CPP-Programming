# STL Linked Lists

---

```cpp
#include <iostream>
#include <list> // Needed for lists


int main()
{
    std::list<char> abc = { 'a', 'b', 'c', 'x', 'y', 'z' };

    for (const auto& elem : abc)
    {
        std::cout << elem << " ";
    }
    std::cout << "\n";

    return 0;
}

```
